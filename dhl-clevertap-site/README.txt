How to publish on GitHub Pages
1. Create a new GitHub repository.
2. Upload index.html and the assets folder to the repository root.
3. In GitHub, go to Settings > Pages.
4. Under Build and deployment, choose Deploy from a branch.
5. Select the main branch and root folder, then save.
6. GitHub will generate a public URL for the page.

Notes
- The site is static and has no external dependencies.
- It is designed to work well as a standalone page and to be easily editable in any code editor.
- The page uses only HTML, CSS, JavaScript and SVG files, so nothing else is required.
